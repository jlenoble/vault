# @organon/prettier-config

Organon's [Prettier](https://prettier.io) config.

## Usage

Edit `package.json`:

```jsonc
{
	// ...
	"prettier": "@organon/prettier-config",
	"devDependencies": {
		// ...
		"@organon/prettier-config": "~1.0.0"
	}
}
```

Install:

```bash
npm install --save-dev @organon/prettier-config
```
